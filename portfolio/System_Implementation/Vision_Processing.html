Vision Processing Implementation

The vision processing captures a side profile of the part to analyze and determine the
orientation of the part. The code is writen python and uses the Open CV and Serial communications
libraries. The first step in the process is to capture the image frame by frame. The image is
then converted to grayscale. Then a threshold is applied to convert the image to black and
white. At this point the part can be see as all black against an all white back ground.

The part is then template matched against four different templates. Each template represents
the part in one of the four different orientations. If the correlation from template matching is
very low then it is assumed that there is no part. Otherwise, the template with the largest
correlation is the part orientation. The proccess runs at around 1 Hz. Each iteration a seial
message is sent to the Arduino with the most recent information on part orrientation.
